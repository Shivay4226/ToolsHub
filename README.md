# 💰 Loan Eligibility Checker

A sleek, responsive, and user-friendly **Loan Eligibility Checker** web tool that allows users to instantly calculate how much loan they are eligible for based on their income, expenses, interest rate, and tenure.

> Part of a collection of finance utilities like SIP Calculator, Currency Converter, and more.

---

## ✨ Features

- 📱 Modern, responsive design (no sliders – input left, output right)
- 🧮 Real-time loan eligibility calculation
- 📊 EMI, principal, interest breakdown
- 🌐 Built using React, Material UI
- 💼 Clean UI/UX for financial tools
